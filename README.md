# Tatsuya Ogawa — Portfolio

Personal portfolio website built with Next.js.

**Live:** https://portfolio-next-nine-smoky.vercel.app

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Animation:** Framer Motion
- **Smooth Scroll:** Lenis
- **Styling:** CSS Modules
- **Font:** Space Grotesk
- **Deployment:** Vercel

## Sections

- **Hero** — Star field with mouse/scroll parallax
- **About** — Skills and background
- **Works** — Project showcase
- **Contact** — Email and GitHub links

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Push to `main` branch triggers automatic deployment via Vercel.
